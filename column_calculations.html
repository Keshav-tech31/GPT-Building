COLUMN DESIGN DETAILS

Given Data:

  • Number of Columns: 77
  • Column Size: 980 mm × 750 mm = 0.98 m × 0.75 m
  • Column Height: 18 m
  • Wall Height: 5.5 m
  • Wall Thickness (including plaster): 31 mm = 0.031 m

Self-Weight of Column:

IS code 456: 2000, IS code 875 (part 1): 1987
Volume of one column: 0.98 × 0.75 × 18 = 13.23 m³
Self-weight per column: 13.23 × 25 = 330.75 kN
Unit weight (density) of reinforced concrete = 25 kN/m³

LOAD CALCULATIONS

Load from Walls on Column:

Volume of wall per meter (including plaster): 5.5 × 0.031 = 0.1705 m³
Wall load per m³: 0.1705 × 18 = 3.069 kN/m
Plaster load on both sides: 5.5 × 0.012 × 20 = 1.32 kN/m
Total wall load per meter: 3.069 + 1.32 = 4.389 kN/m
Total wall load on one column (18 m height): 4.389 × 18 = 79.002 kN

Load from Slab on Column:

Total slab load for 3 floors = 14,388.75 kN
Load on one column: 14,388.75 / 77 = 186.86 kN

Live Load on Column:

IS CODE 875 PART 2 - 1875
Total live load for 3 floors = 11,511 kN
Load on one column: 11,511 / 77 = 149.49 kN

Total Load on One Column:

Sum of all loads on one column: 330.75 + 79.002 + 186.86 + 149.49 = 746.102 kN
Final Load on One Column: The total load on one column = 746.10 kN
